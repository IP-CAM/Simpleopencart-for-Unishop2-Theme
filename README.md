# SimpleOpencart for Unishop2

Адаптация корзины модуля Simple и Filterit под шаблон Unishop2 на движке
opencart 3

#### Как использовать:

- Залить файлы из папок **_catalog_** и **_system_** в соответствующие папки на
  хостинге
- Обновить модификатор в админке
